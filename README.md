---
##  What is ChromaDB?

**ChromaDB** is an open-source embedding database designed for storing, managing, and querying vector embeddings.  
It enables developers to build applications that require **semantic search**, **contextual retrieval**, and **metadata-aware queries** with ease.

---

##  Why ChromaDB?

- **Fast & Scalable** → Handles millions of embeddings efficiently.  
- **Semantic Search** → Go beyond keyword search by finding meaning-based similarities.  
- **Metadata Support** → Attach rich context to records for smarter queries.  
- **Easy Integration** → Works seamlessly with Python and ML workflows.  
- **Open Source** → Community-driven, transparent, and constantly evolving.
